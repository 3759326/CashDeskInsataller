
Btrieve for Windows NT /95 v. 6.15
-----------------------------------------------------------------

Для работы с Btrieve v.6.15 необходимы следующие параметры
аппаратуры и програмного обеспечения:

- Микропроцессор Intel 386,486 или Pentium;
- Windows 95 или Windows NT

Установка Btrieve 6.15
-----------------------------------------------------------------

- Создать каталог BTI на жестком диске;
- Разархивировать в созданный каталог все файлы из архива BTIWIN.ARJ;
- Скопировать файл BTI.INI в каталог Windows;
- Добавить каталог BTI в список каталогов для поиска в команде PATH
- Запустить программу w32mkset.exe для конфигурирования Btrieve.
